# SoftwareLabs Neural Chat Interface (SNCI) v2026

A futuristic AI chat web app that feels alive, responsive, intelligent, and professional.

## Features

- **Neural AI Terminal**: Inspired by sci-fi dashboards and AI research labs
- **Real-time Chat**: Powered by backend API
- **Glassmorphism UI**: Modern, translucent design
- **AI Status Monitoring**: Live status, response times, and model modes
- **Control Panel**: Customize AI personality, response modes, and more
- **Session Management**: Memory toggle, export chat, clear session
- **Futuristic Animations**: Glowing effects, particle backgrounds, smooth transitions

## Tech Stack

- **Frontend**: HTML5, Tailwind CSS, Vanilla JavaScript
- **Fonts**: Orbitron (headings), JetBrains Mono (body)
- **Icons**: Font Awesome
- **Backend**: External API at `https://falling-queen-f2d7.dhimanparas605.workers.dev/api/chat`

## How to Run

1. Open `index.html` in a modern web browser
2. The app will load and connect to the backend automatically
3. Start chatting with the AI!

## Controls

- **Enter**: Send message
- **Ctrl+P**: Toggle control panel
- **Theme Toggle**: Switch between dark/light themes
- **Settings Icons**: Various AI controls in header

## Customization

- Adjust AI personality with the slider in the control panel
- Change response mode (Short/Detailed/Technical)
- Toggle conversation memory
- Export or clear chat sessions
